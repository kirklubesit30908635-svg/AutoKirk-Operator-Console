# Autokirk Operator Console — Landing Page

Governance-first black & gold landing page for Autokirk Operator Console.

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Netlify (GitHub)
1. Push this repo to GitHub.
2. In Netlify, **New site from Git** → choose the repo.
3. Build command: `npm run build`
4. Publish directory: `.next`
5. Netlify will auto-detect the Next.js plugin via `netlify.toml`.

## Brand assets
- Replace `public/logo.png` with your crest as needed.
